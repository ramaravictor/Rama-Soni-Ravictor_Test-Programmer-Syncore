TEST I – PENGETAHUAN DASAR

1. Mengapa setiap bahasa pemrograman itu terdapat framework, apa kegunaan framework?

Framework adalah kumpulan alat, komponen, dan aturan yang membantu programmer dalam membangun aplikasi atau situs web dengan lebih efisien. Setiap bahasa pemrograman memiliki framework yang unik karena setiap bahasa memiliki kebutuhan dan tujuan yang berbeda.

Framework berguna untuk membantu developer dalam mempercepat proses pengembangan aplikasi atau situs web. Dengan framework, sebagian besar kode dasar dan struktur aplikasi sudah tersedia, sehingga developer tidak perlu membuat semuanya dari nol. Framework juga memudahkan dalam pengorganisasian kode, sehingga lebih mudah untuk dipelajari dan dipelihara.

Selain itu, menggunakan framework juga memungkinkan pengembangan aplikasi dengan lebih konsisten dan mudah di skalakan. Dengan framework, developer dapat mengikuti dasar pengembangan aplikasi yang telah tersedia dan membangun aplikasi dengan arsitektur yang lebih baik, sehingga lebih mudah untuk diubah atau dikembangkan lebih lanjut nantinya.

2. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?

  • 'if else' adalah salah satu struktur kontrol dalam pemrograman yang digunakan untuk memeriksa kondisi dan melakukan tindakan yang berbeda tergantung pada kondisi yang terjadi. Jika kondisi bernilai benar (true), maka kode if akan dijalankan, sedangkan jika kondisi bernilai salah (false), maka kode else akan dijalankan.
  • 'while' adalah struktur kontrol dalam pemrograman yang digunakan untuk melakukan perulangan kode selama kondisi tertentu bernilai benar (true). Jika kondisi (syarat) terpenuhi, maka kode dalam while akan terus dieksekusi.
  • 'do while' mirip dengan while, namun perbedaannya adalah do while melakukan perulangan dulu, kemudian memeriksa kondisinya atau sayaratnya. jika masih benar, maka blok kode akan terus dieksekusi.
  • 'switch case' adalah suatu bentuk kondisi percabangan dimana jika suatu kondisi benar(true), maka blok kode bernilai true akan dijalankan.
  • 'for' adalah pernyataan yang digunakan untuk melakukan perulangan yang secara tepat mengetahui berapa kali pengulangan akan terjadi.
  • 'foreach' adalah struktur kontrol dalam pemrograman yang digunakan untuk melakukan perulangan pada setiap elemen dalam sebuah array atau objek.

3. Apa itu constructor, method, class, object, variabel, parameter, function, procedure?

  • Constructor adalah sebuah method yang digunakan untuk menginisialisasi objek pada saat objek dibuat dari sebuah class.
  • Method adalah sebuah fungsi atau prosedur yang didefinisikan di dalam sebuah class dan dapat dipanggil oleh objek yang dibuat dari class tersebut.
  • Class adalah sebuah blueprint atau cetakan yang digunakan untuk membuat objek. Class berisi atribut atau properti dan method yang mendefinisikan perilaku dari objek yang dibuat dari class tersebut.
  • Object adalah sebuah instance dari sebuah class. Object memiliki atribut atau properti dan method yang sudah didefinisikan di dalam class.
  • Variabel adalah sebuah tempat untuk menyimpan nilai atau data yang dapat diubah atau diproses di dalam program.
  • Parameter adalah sebuah variabel yang digunakan dalam definisi sebuah function atau method dan digunakan untuk menerima nilai atau data yang akan diproses di dalam function atau method tersebut.
  • Function adalah sebuah blok kode yang didefinisikan untuk melakukan tugas tertentu. Function dapat dipanggil dari tempat lain di dalam program dan dapat menerima atau mengembalikan nilai.
  • Procedure adalah sebuah blok kode yang didefinisikan untuk melakukan tugas tertentu tanpa mengembalikan nilai.

4. Perbedaan method=”Post” dan “Get”?

Method "POST" digunakan untuk mengirimkan data ke server untuk memprosesnya, sedangkan method "GET" digunakan untuk meminta data dari server.

5. Jelaskan konsep mengenai web service dan fungsi apa yang digunakan untuk memanggil web service client?

Web service adalah sebuah teknologi yang memungkinkan dua aplikasi atau sistem yang berbeda untuk saling berkomunikasi dan bertukar data melalui jaringan, terlepas dari platform atau bahasa pemrograman yang digunakan. Web service memungkinkan aplikasi yang berbeda untuk berkomunikasi secara efektif dengan menggunakan protokol standar seperti SOAP (Simple Object Access Protocol), XML (Extensible Markup Language), dan HTTP (Hypertext Transfer Protocol).

Untuk memanggil web service client, kita dapat menggunakan berbagai macam teknologi, seperti: SOAP, REST, dan RPC (Remote Procedure Call)

6. Jelaskan mengenai MVC dan HMVC !

MVC (Model-View-Controller) adalah konsep arsitektur software yang membagi aplikasi web menjadi tiga komponen utama: Model, View, dan Controller. Model adalah representasi dari data dan logika bisnis dalam aplikasi, View adalah tampilan yang digunakan untuk menampilkan informasi ke pengguna, dan Controller bertanggung jawab untuk menerima input dari pengguna, memprosesnya, dan memperbarui model atau tampilan.

HMVC (Hierarchical Model-View-Controller) adalah konsep arsitektur software yang merupakan pengembangan dari MVC, yang memperluas kemampuan aplikasi dengan membaginya menjadi beberapa sub-modul MVC yang terpisah dan terstruktur secara hierarkis. Dalam HMVC, setiap sub-modul memiliki Model, View, dan Controllernya sendiri, dan dapat berinteraksi dengan modul lainnya. Dengan HMVC, aplikasi dapat diatur lebih baik dan lebih mudah untuk di-maintain. HMVC memungkinkan pengembang untuk menambahkan, memodifikasi, atau menghapus sub-modul tanpa mempengaruhi fungsi modul lainnya.

7. Sebutkan fungsi curl() dalam php!

CURL (Client URL) adalah fungsi dalam bahasa pemrograman PHP yang digunakan untuk membuat sebuah HTTP Request dan juga biasa digunakan untuk mengakses sebuah Web Services atau API. Fungsi curl() memungkinkan pengguna PHP untuk mengakses halaman web atau sumber daya online lainnya dan melakukan tugas seperti mengambil konten halaman web, mengirimkan data ke server web, mengunggah file ke server, dan banyak lagi.

8. Bagaimana menampilkan sebuah element array dalam PHP ?

Untuk menampilkan elemen array dalam PHP, dapat menggunakan pernyataan perulangan seperti for, foreach, atau while. Ketiga cara tersebut akan menghasilkan output yang sama, yaitu menampilkan setiap elemen array pada baris baru. kemudian dapat di pilih cara yang paling sesuai dengan kebutuhan.

9. Perbedaan HTTP dan HTTPS ?

HTTP (Hypertext Transfer Protocol) adalah protokol standar yang digunakan untuk mengirimkan dan menerima data melalui internet. Namun, protokol ini tidak aman karena data yang dikirimkan tidak dienkripsi dan dapat dibaca oleh pihak yang tidak berwenang. Hal ini membuat data yang dikirimkan melalui HTTP rentan terhadap penyadapan atau manipulasi data oleh pihak yang tidak bertanggung jawab.

HTTPS (Hypertext Transfer Protocol Secure) adalah versi yang lebih aman dari HTTP. HTTPS menggunakan protokol SSL (Secure Sockets Layer) atau TLS (Transport Layer Security) untuk mengenkripsi data yang dikirimkan antara klien dan server web. Dengan ini, data yang dikirimkan melalui HTTPS tidak dapat dibaca oleh pihak yang tidak berwenang dan melindungi dari serangan hacking, phishing, dan penyadapan data.

10. Jelaskan Apa itu API? sebutkan contohnya?

API (Application Programming Interface) adalah aturan atau alat yang digunakan untuk membangun perangkat lunak dan aplikasi. API memungkinkan berbagai aplikasi untuk berkomunikasi satu sama lain dan bertukar data secara efisien dan aman. API memungkinkan developer untuk memanfaatkan layanan dan fungsionalitas yang disediakan oleh aplikasi lain, tanpa harus membangun semuanya dari awal. Contohnya penggunaan Google Maps API, digunakan untuk menampilkan peta dan informasi lokasi di aplikasi dan situs web.
